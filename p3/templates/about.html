About Us

Welcome to our Personalized News Website!

We are a team of dedicated students from "Sidharth Govt. College , Nadaun", passionate about leveraging technology to create innovative solutions. As part of our college project, we have developed this personalized news website to provide users with a seamless and engaging news browsing experience.

Our Mission

Our mission is to deliver a user-friendly platform that keeps users informed with the latest news while offering personalized features for an enhanced experience. We aim to make news accessible and engaging for everyone.

Project Objectives

  • User Authentication: Implement secure sign-in and sign-up functionalities to ensure user privacy and data security.
  • Personalized Experience: Allow users to save their favorite news articles for quick access and easy reference.
  • Accessibility: Integrate text-to-speech functionality to enable users to listen to news articles, making the platform more inclusive.
  • Responsive Design: Ensure the website is mobile-friendly and provides a seamless experience across different devices.
  • Modern User Interface: Create a clean and modern UI using Bootstrap and custom CSS for an aesthetically pleasing experience.

Technologies Used

  • Frontend: HTML, CSS, Bootstrap, JavaScript
  • Backend: Python, Flask
  • Database: MySQL
  • APIs: NewsAPI for fetching news articles
  • Libraries: Flask-Mail for sending emails, pyttsx3 for text-to-speech, Flask-MySQLdb for database interaction
